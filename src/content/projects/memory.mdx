A minimal number memory training game — it shows you a sequence of digits for a few seconds, then asks you to recall them. Each level adds more digits.

Built in a single day during a transatlantic flight to scratch a personal itch: the existing apps either added frustrating countdowns, required accounts, or wouldn't let you quickly give up on a round.

Built with **Kotlin Multiplatform** and **Compose Multiplatform**, running natively on Android, iOS, and in the browser as a compiled Wasm app.
